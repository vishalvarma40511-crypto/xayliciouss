# Product Management Guide

Products are managed from `js/products.js`.

Each product contains:

- id
- name
- category
- price
- image
- description
- badge
- featured

To update a product, edit the product object and save the file. To add a new product, copy an existing product object, give it a new unique id, and update its details.

Keep image files inside `assets/images/` and use matching image paths in the product object.
