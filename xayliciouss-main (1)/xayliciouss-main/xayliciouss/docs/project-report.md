# Xayliciouss Premium Bakery Website - Project Report

## 1. Introduction

Xayliciouss is a premium homemade bakery website prototype designed to present desserts in a warm, elegant, and customer-friendly way. The website supports product browsing, WhatsApp ordering, customer service, user dashboard, chatbot assistance, social gallery display, and responsive design.

## 2. Problem Statement

Small homemade bakery brands need a professional digital presence where customers can view products, understand the brand, send inquiries, and place orders quickly. Without a website, customers may depend only on social media messages, which can make browsing, ordering, and support less organized.

## 3. Objectives

- Create a premium bakery website for Xayliciouss.
- Display products, best sellers, reviews, FAQs, and gallery content.
- Enable direct WhatsApp ordering.
- Add customer support forms with instant response messages.
- Add customer dashboard for order history, tracking, and bonus points.
- Include a chatbot popup for customer guidance.
- Make the website responsive, SEO-friendly, and suitable for prototype demonstration.

## 4. Brand Overview

Xayliciouss is a premium homemade bakery focused on brownies, bento cakes, celebration cakes, cupcakes, dessert boxes, cookies, and custom desserts. The brand personality is premium, warm, elegant, homemade, trustworthy, aesthetic, and customer-friendly.

## 5. Technologies Used

- HTML for semantic page structure.
- CSS for responsive premium glass UI design.
- Vanilla JavaScript for product rendering, cart, forms, FAQ, dashboard, chatbot, and interactions.
- Node.js server for prototype data handling.
- JSON file storage for prototype records.

## 9. User Flow

1. Customer opens the website.
2. Customer browses best sellers or menu.
3. Customer searches and filters products.
4. Customer adds products to cart.
5. Customer checks out through WhatsApp.
6. Customer can sign in and view dashboard.
7. Customer can submit inquiries or reports through contact/customer service forms.
8. Customer can use chatbot or floating WhatsApp button for quick guidance.
