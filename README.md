# 🍰 Xayliciouss – Premium Bakery Website

A modern and responsive bakery website built to showcase delicious cakes, pastries, desserts, and bakery products with an elegant user interface. The project provides customers with an easy way to explore products, learn about the bakery, and place orders online.

## 🌐 Live Demo

**Website:** https://xayliciouss-meil.onrender.com/

---

# 📌 Project Overview

Xayliciouss is a full-stack bakery website designed to provide a smooth and attractive online shopping experience. It features a modern design, responsive layout, and user-friendly navigation, making it accessible across desktops, tablets, and mobile devices.

The platform helps bakeries establish a strong online presence while allowing customers to browse products conveniently.

# 🚀 Installation Guide

## Step 1

Download or clone the repository.

```bash
git clone <repository-url>
```

or download the ZIP file from GitHub.

---

## Step 2

Install **Node.js** on your computer.

Download from:

https://nodejs.org/

---

## Step 3

Open **Node.js Command Prompt** or **Terminal**.

Navigate to the project folder.

```bash
cd path/to/xayliciouss
```

Example:

```bash
cd Downloads/xayliciouss
```

---

## Step 4

Install all dependencies.

```bash
npm install
```

or

```bash
npm i
```

---

## Step 5

Start the application.

```bash
npm start
```

or

```bash
npm run dev
```

---

## Step 6

Open your browser and visit:

```text
http://localhost:3000
```

or the port shown in your terminal.
